# Editor Color Picker

A simple mod that functions as a color picker for the Geometry Dash editor.

<img src="logo.png" width="150" alt="the mod's logo" />

# How to use

1. Open the editor.
2. Press the picker button in the edit menu to activate the tool.
3. Use your finger or mouse to select the color you want to grab — the HEX
   code is copied to your clipboard automatically.
4. Press the picker button again to deactivate the tool.

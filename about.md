# Editor Color Picker

A simple mod that lets you pick any color visible in the Geometry Dash editor
and copies its HEX code to your clipboard.

# How to use

1. Open the editor.
2. Tap the picker button in the edit menu to activate the tool.
3. Touch the area whose color you want to grab — the HEX code is copied
   automatically and shown in a notification.
4. Tap the picker button again to deactivate.
